# Sistema de Gestão de Currículos

Aplicação web desenvolvida com Next.js para gerenciamento de currículos de candidatos.

## Tecnologias

- Next.js (App Router)
- Tailwind CSS
- shadcn/ui
- React Hook Form + Yup
- Sonner (notificações)
- React Icons

## Funcionalidades

- Listagem de currículos com busca em tempo real
- Página de detalhes do candidato
- Formulário de cadastro com validação
- Campos dinâmicos para experiências e formações

## Como rodar

```bash
npm install
npm run dev
```

Acesse: http://localhost:3000
